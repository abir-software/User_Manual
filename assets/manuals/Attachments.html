User Manual: Managing Employee Attachments

Feature Reference: REQ-HR-003

Requirement

The system shall allow for the attachment of joining letters and other employee-specific letters within the employee profile/portal.

Purpose

This feature enables authorized users (such as HR officers or administrators) to upload, manage, and track employee-related documents—such as joining letters, contracts, certificates, and other official attachments—directly within the employee profile.

Access Path

Navigation: Go to Employee Module → Employee Profile → Documents Section

Step-by-Step Guide

Step 1: Open Employee Profile

  1. Navigate to the Employee module from the main dashboard.
  2. From the employee list, select and open the profile of the employee for whom you want to add an attachment.

Step 2: Access the Documents Section

On the top section of the employee profile, click on the “Documents” tab to view or manage all existing attachments.

Step 3: Add a New Document

Click the “+ New” button to add a new attachment. Fill in the required fields as described below:

Field NameDescription
Document Number: Enter a unique identification number for the document.
Employee: Auto-selected based on the logged-in user. HR or Admin users can select other employees based on their access rights.
Document Type: Select the appropriate type of document (e.g., Joining Letter, Offer Letter, NID, etc.).
Attachment: Upload the required file (PDF, DOC, or image formats are supported).
Issue Date: Specify the date on which the document was issued.
Expiry Date: Mention the document expiry date, if applicable.
Notification Type: Choose how the system should notify you regarding document expiry (on expiry date, before few days, everyday till expiry date, on and after expiry).

Step 4: Save or Discard Changes

After filling in all fields, click “Save” to store the record. The system may also save automatically upon closing the form. To cancel changes, click the ‘X’ button to discard all unsaved data.

Step 5: Import or Export Records

Click “Import Records” to bulk upload document records. Click “Export Records” to download existing records for reporting or backup purposes.

Step 6: View and Edit Documents

To view details of any uploaded document, click on the desired document record from the list. You can edit document information directly from this view if you have sufficient access rights.

Step 7: Duplicate or Delete Documents

Click on the Settings (⋮) menu of the selected document. Choose Duplicate to create a copy of the document, or Delete to remove it permanently.

Step 8: View Document Summary

The Documents section at the top of the employee profile displays the total number of documents attached to that employee. Each document can be accessed directly by clicking the count indicator.

End Result

After completing the above steps, the system successfully stores the employee’s attached documents within their profile. HR personnel and authorized users can easily manage, view, and track all attachments in one place.
